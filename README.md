## Acerca de
Este proyecto tiene como objetivo hacer una app de escritorio usando Electron y JS

### Project setup
```
npm install
```

### Start
```
npm run start
```

### Compiles and minifies for production
```
npm run publish
```

### Lints and fixes files
```
npm run lint
```
